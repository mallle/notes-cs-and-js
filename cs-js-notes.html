CS Basics

Paths

Paths are used to reference other files in a filesystem. For example used to referencing an image, a .js or .css file

Relative Paths

Relative paths have the current directory as a starting point.

Reference a file in the current folder:

src="./filename.js"

References a file in a folder above the current folder:

src="../filename.js"

Absolute Paths

Absolute paths start at the root directory (the top-level directory) of a system.

Reference a file on a server:

src="https://www.mordbeitisch.de/_nuxt/img/3221c86.jpg"

Reference a file on your local system (This file is located in the Users directory):

src="/Users/mordbeitisch.jpg"

Terminal/Commands (MAC)

A small list of useful commands for the terminal.

Show current directory:

~ pwd

Change directory:

~ cd name-of-directory

Go back one level/directory:

~ cd ..

List of directories and files in the current directory:

~ ls

List of directories and files in the current directory as a long ist format:

~ ls l

Make a new directory:

~ mkdir name-of-directory
